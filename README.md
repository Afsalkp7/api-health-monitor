# CloudStack API Monitor - Frontend

The user interface for the CloudStack monitoring system. Built with Next.js 14 (App Router), this dashboard provides real-time visualization of API health, detailed latency graphs, and insight reports.

## Tech Stack

* **Framework:** Next.js 14 (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Authentication:** NextAuth.js (Custom Credentials Provider)
* **Visualization:** Recharts (Area/Line charts)
* **Icons:** Lucide React
* **HTTP Client:** Axios


## Key Features Implemented

### 1. Authentication & Onboarding
* **Custom Login Flow:** Integration with Backend JWT.
* **Registration:** Multi-step registration flow including OTP entry.
* **Session Management:** NextAuth handles session persistence and auto-refreshing of tokens.

### 2. Real-Time Dashboard
* **Status Indicators:** Color-coded badges (Green/Red) showing live status of all monitors.
* **Optimistic UI:** Immediate UI updates when pausing/resuming monitors or deleting items, providing a snappy experience.
* **Skeleton Loading:** Custom skeleton states to prevent layout shift (CLS) during data fetching.

### 3. Deep-Dive Monitor Details
* **Interactive Graphs:** Visualizes response time history using gradient-filled Area charts.
* **Visual Logic:** Graph lines turn red dynamically during "DOWN" periods.
* **Detailed Logs:** A table view of the last 20+ pings with timestamps and status codes.
* **Stats Cards:** Displays calculated 24h Uptime, Avg Latency, and Last Checked time.

### 4. Insights & Analytics View
* **Health Score:** A high-level system health overview (Excellent, Degraded, Critical).
* **Anomaly Reporting:** dedicated UI for viewing alerts generated by the backend analytics engine (e.g., "Performance Degradation").

### 5. Settings & Profile
* **Profile Management:** Ability to update Display Name.
* **Security:** Password change functionality requiring current password verification.
* **Read-Only Fields:** Email is locked to ensure account security and data integrity.
